Create automatic plan links from your event blog so that your users can make plans on Planyp.us.

== Description ==

For blogs that publish events, let your users tell their friends about your event and join together online
to make plans to attend, at Planypus (http://planyp.us)
By default, post timestamp is used as scheduled time, and "planypus_event_location" custom field as street address, and "planypus_event_location_name" as the name of venue.
If you are using the Event Calendar 3 plugin (http://wpcal.firetree.net), install planypus_links_ev3.php to use the event schedulefor the date/time.
Support for  calendaring plugins can be easily implmented.  See "Planypus_links_ev3.php" for an example.

See php files for more info.
== Installation ==

1. Upload `planypus_links.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the "Plugins" menu in WordPress
3. If uisng Event Calendar 3 plugin (http://wpcal.firetree.net), upload and activate "planypus_links_ev3.php"
4. If needed, When creating a post, add "planypus_event_location" custom field with a one-line street address.
